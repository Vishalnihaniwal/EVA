What is the Start Node Option?




This option is selected to remember the option inputs selected in the previous runs of the project i.e. it stores the data of Systems Required option, Select Config File option, Select TestCase File Option, all the IP, Ports and Browser Selections of Node, Image Comparison Option and Execution Result Location field.

It stores all this data in a Json object and displays the result by fetching the Json Object and populating all fields on the start of the UI.
