What is the Start Server Option?




Start Server starts the hub or the server to which the nodes have to be connected. This is done in order to execute the script on the respective nodes.


On clicking start server, the command prompt gets invoked and the jar file of the hub starts executing. If the server is already executing, it displays an alert box asking if we want to kill the previous one. On clicking yes, the previous process is killed and a new execution starts , otherwise on clicking No and Cancel, the previous process is retained.
